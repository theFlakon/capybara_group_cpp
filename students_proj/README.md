# str_program

**student_proj** is an **educational console application written in C++**.  
The project demonstrates basic classes usage and file input/output.

## Build and Run

The project is built using **CMake**.

### Build

From the project root directory:

```bash
cmake -S . -B build
cmake --build build
```

### Run

After a successful build, the executable is located at:

```text
build/bin/program
```
